Tera Byte Video Game Creation Camp

Courses

We've been teaching video game creation for 9 years. Now, you can learn to program your own games for the web.

Beginning Game Creation

This course is for anyone and everyone who is interested in video game creation. Campers learn Multimedia Fusion 2, an easy to use program for creating video games, screen savers, and exciting applications. Campers will learn the basics of the program by making video games such as Pong, a Mario-style platform game, a street fighter, and a flight simulator.

At the end of the week, the campers will get a CD containing all of their work in a format playable on their home PC, even without having to purchase the program.

Advanced Game Creation

This course is for returning campers. We continue to use Multimedia Fusion 2. Campers will learn more advanced features of the program by making super advanced video games with way more than you learn in the beginning class.

If a camper repeats the course, he or she will become more advanced and learn to make more exciting, fun games. There is always more kids can do and learn, so they can take this class many times and continue to build upon their skills by making any game of their choosing.

At the end of the week, the campers will get a CD containing all of their work in a format playable on their home PC, even without having to purchase the program.

Intro to Programming

Learn Javascript programming using the software from CodeHS.

Starting with teaching Karel the Dog tricks, you learn the fundamentals of programming and build a foundation to learn and understand how to write real programs that run on the web!

Curriculum designed by Tera Byte founder, Zach Galant, and Jeremy Keeshin based on their experience teaching for 3 years at Stanford and the past year to thousands of students aged 5 to 85.
